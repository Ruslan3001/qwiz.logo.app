# Car Brand Quiz 🚗❓

Простое и быстрое веб-приложение (Standalone Web App) для проверки знаний автомобильных марок. Написано на чистых HTML, CSS и JavaScript без использования тяжелых фреймворков.

## 🌟 Особенности
* **Два режима игры:** Угадывание марки по векторной эмблеме или по реальной фотографии автомобиля из сети.
* **Статистика в реальном времени:** Подсчет правильных ответов и встроенный таймер.
* **Адаптивный дизайн:** Отлично работает как на десктопе, так и на мобильных устройствах.
* **Web App (PWA-lite):** Возможность установки на главный экран смартфона (iOS/Android) для использования в полноэкранном режиме, как нативное приложение.

## 🚀 Как запустить

Поскольку приложение состоит из одного HTML-файла, установка максимально проста:
1. Скачайте репозиторий или просто файл `index.html`.
2. Откройте `index.html` в любом современном браузере.

**Для использования на смартфоне (через GitHub Pages):**
1. Перейдите по ссылке проекта (например, `https://Ruslan3001.github.io/qwiz.logo.app/index.html`).
2. В браузере (Safari/Chrome) выберите опцию «Поделиться» -> «На экран "Домой"».
3. Запускайте викторину прямо с рабочего стола!
